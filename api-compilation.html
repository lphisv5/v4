Overview

Public
ℹ All APIs on this page are completely free and require no API key or user authentication. Simply send a request to the desired endpoint and you will receive a response immediately.

⚠ APIs marked Offline are currently unavailable or under maintenance. Join our Discord server for real-time status updates. Please avoid sending excessive repeated requests to any single endpoint.

AI APIs

Online
🔑
UUID Generator
Generate universally unique identifiers (v4)
GET Online

Returns a freshly generated UUID v4 string on every request. Suitable for use as unique identifiers in databases, session tokens, transaction IDs, or any application requiring globally unique values.

Endpoint
GET https://generators-uuid.vercel.app/api/uuid
🤖
AI Free Chat
Open-access AI chat completions endpoint
POST Online

A free, donation-funded AI chat completion API built on OpenRouter. Send a message in the request body and receive an AI-generated reply. No authentication or API key is required by design.

Endpoint
POST https://yanz-ai-free.vercel.app/api/chat
Request Body (JSON)
ParameterTypeRequiredDescription
message string Required The user message or prompt to send to the AI model.
model string Optional Model identifier. Defaults to a free-tier model when omitted.
💬
Chatbot AI
Conversational AI via URL query parameters
GET Online

A GET-based AI chat endpoint. Pass your prompt and optionally a model identifier directly in the URL query string — ideal for quick integrations, webhooks, Discord bots, or any browser-based requests where POST requests are not practical.

Endpoint
GET https://yanzai-bot.vercel.app/api/chat
Query Parameters
ParameterTypeRequiredDescription
prompt string Required URL-encoded message or question for the AI to respond to.
model string Optional Model selector. Pass auto-free to use the default free-tier model.

Roblox APIs

Online
🕹
Roblox Status
Real-time Roblox platform & API health check
GET Online

Returns the current operational status of the Roblox platform. Two endpoints are available: one that checks the Roblox game platform's health, and one that checks this API service's own availability.

Endpoint — Roblox Platform Status
GET https://api-roblox-xi.vercel.app/api/roblox/status
Endpoint — API Service Health
GET https://api-roblox-xi.vercel.app/api/status
📰
Roblox News
Latest news and announcements from Roblox
GET Online

Fetches the most recent news articles and official announcements published by Roblox. Returns structured news data including titles, summaries, and publication timestamps — useful for bots, dashboards, or news aggregators.

Endpoint
GET https://robloxnews.vercel.app/api/robloxnews
📦
Roblox Version
Current Roblox client version by platform
GET Online

Returns the latest Roblox client version number for a specified platform. Supports iOS, Android, Windows, and macOS. Useful for detecting outdated clients or automating platform-specific update checks.

Endpoint
GET https://yanz-api.vercel.app/api/version?platform={platform}
Query Parameters
ParameterTypeRequiredAccepted Values
platform string Required ios  ·  android  ·  windows  ·  macos
All Platform Endpoints
GET https://yanz-api.vercel.app/api/version?platform=ios
GET https://yanz-api.vercel.app/api/version?platform=android
GET https://yanz-api.vercel.app/api/version?platform=windows
GET https://yanz-api.vercel.app/api/version?platform=macos

Executors API
Executor list and metadata for Roblox scripting
GET Online

Returns a structured list of Roblox script executors along with their associated metadata such as platform support, version, and current status. Useful for community tools, comparison dashboards, or any project tracking executor availability.

Endpoint
GET https://executors-one.vercel.app/executors
